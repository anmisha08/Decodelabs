# UI/UX Internship Task 1

## Project Title
Fitness App UX Case Study

## About the Project
This project focuses on understanding the challenges faced by busy office professionals who struggle to maintain fitness routines due to time constraints and work pressure.

The UX process includes:
- User Persona
- Empathy Mapping
- User Pain Points
- Goals & Motivation Analysis

## User Persona
The target user is a working professional who wants to stay fit but lacks time for regular exercise because of office workload and long working hours.

## Empathy Phase
The empathy mapping process helped in understanding:
- What users think and feel
- Their frustrations and motivations
- Daily habits and challenges
- Fitness-related pain points

## Tools Used
- Figma

## Files Included
- UX Research PDF
- User Persona Design
- Empathy Map

## Objective
To create a user-centered fitness app experience that motivates busy professionals to maintain healthy habits through simple and effective solutions.
